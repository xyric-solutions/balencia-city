## Exterior Architecture

### Description
The Communication Hub is the most luminous structure in Balencia City -- a cluster of 3-4 interconnected tower pods linked by glass-enclosed sky-bridges and transparent communication tube conduits. Each tower pod is a sleek vertical form, slightly different in height (tallest 30 floors, others 25-28), giving the cluster an asymmetric but balanced composition. Signal light strips visibly travel between the towers along the conduit tubes, creating constant visual movement in the connections.

Each tower top carries antenna arrays -- thin vertical spikes and small dish forms that give the crown a bristling, broadcast-ready profile. Large display screens are mounted on the outward-facing facades of each pod, showing abstract communication wave patterns. The ground level opens into a shared plaza connecting all tower bases, creating an inviting public gathering space beneath the soaring connected structures. A satellite dish or broadcast element crowns the main (tallest) tower. The entire complex radiates more light than any other district structure due to using SIA's own burnt orange as its accent color.

### Key Architectural Elements
- Main tower pod: tallest at 30 floors, sleek vertical form, most display screens
- Secondary tower pods: 2-3 additional towers at 25-28 floors each
- Glass-enclosed sky-bridges between towers: 2-3 bridges at different heights
- Communication tube conduits: transparent tubes running between pods (signal light animation)
- Antenna arrays on each tower top: thin vertical spikes + small dish forms
- Signal light strips traveling between towers along conduits (emissive moving elements)
- Large display screens on outward facades: 1-2 per pod (flat rectangles, emissive)
- Ground-level open plaza: shared base connecting all tower bases
- Satellite dish on main tower crown: parabolic reflector form
- Pipeline hardpoint on main tower for SIA energy conduit

### Silhouette Markers
Multiple connected towers -- the only multi-pod structure in the city (Career is also a cluster but has different proportions). The visible sky-bridges and communication conduits between distinct vertical pods create a web-like upper profile. Antenna bristle at each crown distinguishes it from Career's clean observation decks.

## Interior -- Communication Nexus

### Description
The interior of the main tower pod ground floor opens into a communication nexus -- a circular space designed around the concept of visible conversation. Holographic conversation threads manifest as floating light ribbons that arc between active stations, showing communication flow as physical light paths. Voice/video calling towers stand at 4 cardinal points of the room -- tall cylindrical booths with life-size holographic projections of remote participants visible through translucent walls.

A central group collaboration zone features holographic whiteboards -- large flat translucent surfaces that float at varying angles, covered with abstract notation and diagrams. Messages are visible as glowing particles traveling along guided paths between stations, creating a constant gentle flow of light through the space. The ceiling is an open lattice showing the underside of the sky-bridge connection to the adjacent pod.

### Focal Point
The holographic conversation threads -- floating light ribbons arcing across the room between active communication stations, creating a visible web of connection above the collaboration zone.

### Supporting Props (6 items)
- Voice/video calling towers: 4 cylindrical booths at cardinal points with translucent walls

- Message particle paths: curved tubes at ceiling height guiding glowing particles between stations
