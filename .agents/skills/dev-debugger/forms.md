# DEV-03 Forms: Templates & Checklists

Loaded on demand for output templates, quality checklists, and validation criteria.

---

## Bug Report Template

```markdown
## Bug: [Brief Description]

### Expected vs Actual
**Expected:** [What should happen]
**Actual:** [What happens]
**Error:** `[Exact error text]`

### Reproduction Steps
1. [Step 1]
2. [Step 2]
3. [Bug occurs]

### Environment
- OS: [e.g., macOS 14.0]
- Node/Python: [version]
- Relevant packages: [versions]

### Hypothesis
1. [Most likely cause]
2. [Alternative cause]

### Fix Applied
[Description + code change]

### Verification
- [ ] Bug no longer occurs
- [ ] No regression
- [ ] Test added to prevent recurrence
```
